======================================================================
OPEN-R Version 1.1.3 FlashUpdater for ERS-210 
======================================================================

   本プログラムは、AIBO ERS-210に内蔵されているOPEN-Rソフトウェアを、
   バージョン1.1.3に更新します。

   内蔵OPEN-Rソフトウェアのバージョンが1.1のERS-210上で、OPEN-R SDKで
   作成されたソフトウェアを動作させるには、最初に内蔵OPEN-Rソフトウェ
   アを更新してください。

      [ご注意]

         ERS210のコアユニットに「OPEN-R Ver 1.1.2」または「OPEN-R Ver
         1.1.3」と書かれたシールが貼られている場合は、更新の必要はあり
         ません。

OPEN-Rソフトウェアの更新手順

   1. バッテリーを充分に充電します。AIBOのバッテリー残量が少ないと、更
      新が失敗することがあります。

   2. 空のAIBOプログラミングメモリースティックに、OPEN-Rディレクトリを
      コピーします。

       [ご注意]

          AIBOプログラミングメモリースティック以外のメモリースティック
          は使用しないようにしてください。使用した場合、更新は行なわれ
          ず、さらに、AIBOを活動停止状態にできなくなります。AIBOを活動
          停止状態にできなくなったときは、メモリースティックを取り出し
          た後、「トラブルシューティング」を参照してバッテリーを取り出
          してください。再びバッテリーを挿入すれば、AIBOが正常に起動し
          ます。

   3. AIBOに、AIBOプログラミングメモリースティックを挿入します。

   4. 胸のポーズボタンを押します。

      胸ランプが緑色に点灯し、約10秒後に、更新中であることを表すブザー
      音が鳴り始めます。このブザー音が鳴り終わり、AIBOが活動停止状態
      （胸ランプが消灯）になったら更新終了です。更新は、約1分で終了し
      ます。

      更新中は、絶対にメモリースティックやバッテリーを取り出さないでく
      ださい。

トラブルシューティング

   - ポーズボタンを押しても、約10秒すると短いブザー音が鳴り、すぐに活
     動停止状態になる。

      <原因と対策>
         AIBOの内蔵OPEN-Rソフトウェアは、すでにバージョン1.1.3です。更
         新の必要はありません。

   - ブザー音が断続的に鳴り続け、胸ランプが緑に点減する。

      <原因と対策>
         ERS-210以外のAIBOに対して本プログラムを使用しているか、または、
         内蔵OPEN-Rソフトウェアが1.1.3よりも新しいバージョンです。ポー
         ズボタンを押してAIBOを活動停止状態にしてください。

   - ブザー音が断続的に鳴り続け、胸ランプが緑とオレンジ交互に点滅する。

      <原因と対策>
         何らかのエラーが発生しています。最も考えられるのは、バッテリー
         の充電不足です。ポーズボタンを押して活動停止状態にした後、バッ
         テリーを充分に充電してください。

   - ポーズボタンを押しても何も起こらず、もう一度ポーズボタンを押して
     も活動停止状態にすることができない。

      <原因と対策>
         AIBOプログラミングメモリースティックの内容が壊れているか、
         AIBOプログラミングメモリースティック以外のメモリースティック
         を使用しています。バッテリーパックのロックを解除し、バッテリー
         を取り出してください。バッテリーパックのロックを解除するには、
         バッテリーパックロック解除用穴に、細いピンなどを差し込みます。
         （「バッテリーパックロック解除用穴の位置については、AIBO
         ERS-210本体の取扱説明書をご覧ください。）

   - バージョンアップ後、ポーズボタンを押すと、悲しげなメロディーを出
     し、すぐに活動停止状態になった。

      <原因と対策>
         何らかの理由で、更新が中断されました。再度更新を行なってくだ
         さい。
